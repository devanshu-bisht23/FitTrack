# FitTrack Backend

This is the backend API for the **Fitness Tracker Application**.
It is built using **Strapi**, which provides a headless CMS and REST APIs for managing workout and fitness data.

The backend handles storing and managing data used by the frontend application.

---

## Tech Stack

* Strapi
* Node.js
* PostgreSQL
* REST API

---

## Installation

Clone the repository:

git clone https://github.com/your-username/fitness-tracker.git

Navigate to the backend folder:

cd backend

Install dependencies:

npm install

---

## Running the Backend

Start the development server:

npm run develop

The server will run at:

https://cozy-laughter-713b0b24a1.strapiapp.com/

---

## What This Backend Manages

The backend manages data related to:

* workouts
* exercises
* user fitness tracking
* activity logs

These APIs are consumed by the **React frontend application**.
