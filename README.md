# FitTrack

Fitness Tracker is a full-stack web application that helps users log workouts, track exercises, and monitor their fitness activity.

The project consists of a **React frontend** and a **Strapi backend** that manages workout data through APIs.

---

## Tech Stack

Frontend

* React
* TypeScript
* Vite

Backend

* Strapi
* Node.js
* REST API

Tools

* Git
* npm

---

## Running the Project:

### 1️⃣ Start Backend

Navigate to the server folder:

cd server

Install dependencies:

npm install

Run Strapi backend:

npm run develop

Backend runs at:

https://cozy-laughter-713b0b24a1.strapiapp.com

---

### 2️⃣ Start Frontend

Navigate to client folder:

cd client

Install dependencies:

npm install

Run frontend:

npm run dev

Frontend runs at:

https://fit-track-client-chi.vercel.app

---

## Features

* Track workouts and exercises
* Manage fitness activity data
* Track live steps 
* Backend API for storing workout information
* Clean and responsive UI
