# FitTrack 💪

FitTrack is a web application that helps users track their workouts, exercises, and overall fitness progress.
The goal of this project is to provide a simple interface where users can log workouts and monitor their activity over time.

---

## 🚀 Features

* Add and manage workouts
* Track exercises and fitness activity
* View workout history
* Simple and responsive UI
* Fast development setup using Vite

---

## Tech Stack

Frontend:

* React
* TypeScript
* Vite
* CSS / Tailwind

Tools:

* ESLint
* Node.js
* npm

---

## Installation

Clone the repository:

git clone https://github.com/your-username/fitness-tracker.git

Go into the project directory:

cd fitness-tracker

Install dependencies:

npm install

---

## Running the Application

Start the development server:

npm run dev

The application will run at:

https://fit-track-client-chi.vercel.app/

---

## Purpose of the Project

This project was built to practice modern frontend development using **React and TypeScript**, while also creating a useful application for tracking workouts and fitness progress.
